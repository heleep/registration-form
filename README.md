# Vendor Registration Portal otp based system

## 📌 Project Overview

This project is a Vendor Registration Portal where users can fill out a registration form and download their submitted details as a PDF.

---

## 🚀 Features

* Vendor registration form
* Clean and professional UI
* Data capture and processing
* PDF download functionality
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Python (Flask)
* HTML5
* CSS3
* JavaScript
* PDF generation libraries

---

project/

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/vendor-registration.git
```

### 2. Navigate to project folder

```
cd vendor-registration
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run the application

```
python app.py
```

---

## 🌐 Usage

1. Open browser and go to:

```
http://127.0.0.1:5000/
```

2. Fill the registration form
3. Submit and download PDF
